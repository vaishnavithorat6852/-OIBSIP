# Level 2 - Task 1: Calculator

A fully functional, highly aesthetic browser-based calculator featuring a glassmorphic user interface. Created as part of the **OASIS Infobyte Summer Internship Program (OIBSIP)**.

## Project Demo
- **Live Preview:** [View Project](https://vaishnavithorat6852.github.io/-OIBSIP/OIBSIP/WebDevelopment-Level2-Task1-Calculator/)

## Features
- **Modern UI:** Glassmorphism theme with a vibrant backing radial glow, clean Outfit typography, and glowing button active states.
- **Sequential Evaluation (Operator Chaining):** Allows sequential calculations (e.g. `5 + 3 × 2 = 16`) without needing intermediate resets.
- **Float Precision Control:** Solves floating point issues (e.g., `0.1 + 0.2` results in exactly `0.3` instead of `0.30000000000000004`).
- **Safety Guards:** Displays a clean `Error` message for division by zero and prevents double decimal entries.
- **Keyboard Support:** Supports standard numerical inputs, backspace, operators, and enter/equals, complete with visual press animation feedback.
- **Responsive Layout:** Grid alignment adjusts dynamically for mobile and desktop viewports.

## Tech Stack
- **HTML5:** Semantic layouts and screen structure.
- **CSS3:** Custom styles, CSS Grid layout, and animations.
- **Vanilla JavaScript:** Core calculation state machine and event handling.

## Running Locally
1. Clone this repository:
   ```bash
   git clone https://github.com/vaishnavithorat6852/-OIBSIP.git
   ```
2. Navigate to the project folder and open `index.html` directly in any web browser, or start a local server:
   ```bash
   cd -OIBSIP/OIBSIP/WebDevelopment-Level2-Task1-Calculator
   python -m http.server 8000
   ```
3. Open `http://localhost:8000` in your web browser.
