# Level 2 - Task 2: Tribute Page

A visually engaging, responsive tribute page dedicated to **Dr. A. P. J. Abdul Kalam**, the 11th President of India and the "Missile Man of India." Developed as part of the **OASIS Infobyte Summer Internship Program (OIBSIP)**.

## Project Demo
- **Live Preview:** [View Project](https://vaishnavithorat6852.github.io/-OIBSIP/OIBSIP/WebDevelopment-Level2-Task2-TributePage/)

## Features
- **Design System:** Serif typography (`Playfair Display`) for headings and clean sans-serif typography (`Outfit`) for body copy to provide an elegant, readable contrast.
- **Section Color Contrasts:** Alternates between dark slate gradients and light/pastel lavender backgrounds to structure content.
- **Responsive Layout:** Grid layout adapts cards and text padding smoothly across mobile, tablet, and desktop viewports.
- **Key Timeline:** Displays his journey and achievements in structured timeline cards with subtle hover shift animations.
- **Tribute Quote:** Features one of his signature quotes styled in a dark callout banner.

## Tech Stack
- **HTML5:** Semantic components (`<header>`, `<main>`, `<section>`, `<article>`, `<footer>`).
- **CSS3:** Custom styles, Google Fonts imports, CSS Grid layouts, and hover transitions.

## Running Locally
1. Clone this repository:
   ```bash
   git clone https://github.com/vaishnavithorat6852/-OIBSIP.git
   ```
2. Open `index.html` in your web browser, or launch a local server:
   ```bash
   cd -OIBSIP/OIBSIP/WebDevelopment-Level2-Task2-TributePage
   python -m http.server 8000
   ```
3. Open `http://localhost:8000` in your web browser.
